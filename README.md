## This is a Student Database Management System for school students using C

# Up till now:
> has name, class, address, rollNo and percentage(marks) datas for a student record
> Implemented addRecord(), searchRecord(), deleteRecord() and displayRecord() functions
> addRecord() - adds every data of a student record
> searchRecord() - searches records by rollNo and get all information of particular rollNo
> deleteRecord() - deletes record by rollNo
> displayRecord() - all the records will be displayed

# Future Updates:
> for searchRecord(), combination of class-rollNo, class-Name, etc,.
> for displayRecord(), display only specific records by rollNo or name, display records of a specific class
> sort and display record by rollNo, class (ascending/descending)
